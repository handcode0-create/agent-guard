agent-guard

Ne laisse plus un agent IA réécrire silencieusement ton code.
agent-guard force tous les changements générés par l'IA à passer par un dossier de staging et une validation humaine explicite.

Voir sur GitHub Comment ça marche ?

TL;DR

L'agent IA n'a jamais accès directement à ton projet réel.

Il écrit ses fichiers dans .agent-guard/staging/, tu vois les diffs, puis tu décides quoi approuver ou rejeter. Aucune écriture automatique, aucun commit surprise. Si tu utilises des agents pour modifier ton code, teste-le avant que ça ne casse tout en silence.

Cas d'usage typique

Tu demandes à Google AI Studio ou un autre agent de refactorer ton front (React, Tailwind) ou de générer du code backend.

  • L'agent écrit dans .agent-guard/staging/, pas dans ton repo.
  • Tu inspectes avec agent-guard diff.
  • Tu appliques uniquement ce que tu valides avec approve.

Bridge Google AI Studio

Une extension Chrome redirige les fichiers générés par Google AI Studio vers le staging d'agent-guard.

  • Aucun fichier modifié en direct dans ton projet.
  • Compatible avec ton workflow actuel dans AI Studio.

Scénario de test rapide

Tu peux tester le concept en quelques minutes sur un projet de démo. 

# 1. Créer un projet de test
mkdir demo-agent-guard
cd demo-agent-guard
npm init -y
mkdir src
echo "console.log('Hello');" > src/app.js

# 2. Initialiser agent-guard
agent-guard init

# 3. Simuler un agent IA (écriture dans le staging)
cp src/app.js .agent-guard/staging/src/app.js
echo "console.log('Hello from AI');" >> .agent-guard/staging/src/app.js

# 4. Voir les modifications proposées
agent-guard status
agent-guard diff src/app.js

# 5. Appliquer seulement ce que tu acceptes
agent-guard approve src/app.js
# ou rejeter
# agent-guard reject src/app.js

Pourquoi pas un simple hook Git ?

Un hook Git peut être oublié, contourné ou désactivé.

Avec agent-guard, l'agent n'a tout simplement jamais accès au projet réel : il ne peut écrire que dans un dossier physique séparé. La validation humaine devient structurelle, pas optionnelle.
